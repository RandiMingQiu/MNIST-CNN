# MNIST 手写数字识别项目
基于CNN的 MNIST 手写数字分类入门项目

## 📋 项目介绍
本项目是深度学习入门经典实践，使用 MNIST 手写数字数据集完成模型训练与数字预测。
适合零基础入门计算机视觉、神经网络的学习项目，代码简洁易懂，可直接运行。
(一个大一新生的深度学习入门)
## 📊 数据集说明
- **data**：MNIST 手写数字数据集
- **阿拉伯数字**：0~9 共 10 类手写数字图片
- **数据规模**：60000 张训练图片 + 10000 张测试图片
- **图片规格**：28×28 像素灰度图
